# ShuTong Protocol（书童协议）

> **Version**: V0.1 | **版本**：V0.1
> **Date**: 2026-07-20
>
> ShuTong Protocol is a cognitive-layer human-AI collaboration protocol.
> Core thesis: AI is not an answer machine, but a cognitive calibrator.
>
> 书童协议是认知层的人机协作协议。
> 核心主张：AI 不是答案机器，是认知校准器。
