# 概率确定 / Probabilistic Determinism

> 书童协议的认识论基础：概率性不是认知的缺陷，是认知的固有属性。

## 核心立场

**不是"选择行为可概率，信息传递必确定"这么简单的二分。**

核心认识论立场：**概率性不是认知的缺陷，是认知的固有属性。**

人脑决策也是概率性的——神经元随机放电，突触权重波动。
你选择喝可乐而不是雪碧时，不会说"我的 confidence 只有 0.73，请你校准一下"。
你的选择就是你当前认知状态的确定性表达。

模型同理。它的概率采样不是"不确定的信号"，而是它在当前权重和上下文下的确定性认知结果。

## 为什么拒绝置信度

让模型输出 `{"intent": "stop", "confidence": 0.85}`，等于让模型做元认知——"我猜自己有多确定"。但：

1. 模型的 confidence 和实际准确率经常脱钩
2. 7B 小模型的 confidence 更不可靠
3. **置信度本身也是概率性输出**——用不确定性的不确定性来降低不确定性，这是无限回归

书童协议终止这个回归：
- 模型选择 intent="stop" → 这就是它的判断
- 规则层校验条件 → 确定性执行
- 错了？审计日志记录 → 下次迭代修正 prompt

**不是"因为模型不可靠所以不用置信度"，是"因为概率性本身就是认知的固有属性，所以不需要置信度"。**

## 概率确定 vs 置信度校准

| 维度 | 置信度校准（传统做法） | 概率确定（书童协议） |
|------|---------------------|-----------------|
| 概率观 | 概率 = 不确定性 = 需要被消除 | 概率 = 认知的固有属性 = 不需要被消除 |
| 模型角色 | 概率生成器，需要人类校准 | 认知主体，概率采样即确定选择 |
| 置信度 | 需要模型自我评估（confidence score） | **拒绝置信度**，选择即判断 |
| 决策链 | 模型输出 → confidence → 人类决定是否采纳 | 模型选择 → 规则层校验 → 执行 |
| 错误处理 | confidence 过滤（低 confidence 不执行） | 规则层兜底 + 审计日志 + 迭代修正 |
| 哲学立场 | 人类是最终决策者 | 人类是校准者，模型是认知参与者 |

**关键区分：**

置信度校准把模型视为"不可靠的概率生成器"，需要人类或系统评估"它有多确定"。
概率确定把模型视为"有认知主体的决策参与者"，它的概率采样就是它当前认知状态的确定性表达。

修正方式不同：
- 置信度校准：调阈值、加过滤、让人类审查低 confidence 输出
- 概率确定：规则层兜底、审计日志记录、迭代修正 prompt 或微调模型

前者是**工程补丁**，后者是**协议设计**。

## 三层确定性

```
┌─────────────────────────────────────────┐
│  选择层 —— 概率性（对模型而言是确定性表达） │
│  · 模型生成认知显化（自然语言采样）         │
│  · 生成建议靶子（多样性采样）              │
│  · 生成追问措辞（创造性采样）              │
│  · 意图识别（概率采样即确定选择）           │
│  允许：模糊、多样、试探、不完美             │
│  但：采样结果就是模型的确定性判断，无需二次评估 │
└─────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────┐
│  规则层 —— 确定性                        │
│  · 结构检验：LLM输出是否含必要结构         │
│  · 来源标注检验：假设来源是否为"纯猜测"    │
│  · 轮次提醒：≥N轮时提示用户可喊停         │
│  · 执行拦截：未确认→禁止执行              │
│  不允许：概率判断、模型自治决策             │
└─────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────┐
│  信息层 —— 绝对确定性                      │
│  · 用户确认的事实（原文记录+用户原话）     │
│  · 追问-确认链（时间戳+来源+原话对照）     │
│  · 产品包（锁定后不可变）                  │
│  不允许：压缩、失真、推测                  │
└─────────────────────────────────────────┘
```
